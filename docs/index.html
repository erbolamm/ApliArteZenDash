ATENCIÓN: CERRARÁ APLICACIONES

Esta herramienta está diseñada para exprimir cada gota de rendimiento cerrando de golpe procesos en segundo plano.

Si vas a entrar en modo Streaming o Gaming, asegúrate de que no estás usando activamente Dropbox, Chrome o aplicaciones que requieran sincronización en la nube en ese momento.

✅ Lo recomendado

  • Comprueba que todo trabajo esté guardado antes de hacer "click".
  • Ejecuta el .hta haciendo doble click sin necesidad de instalar.
  • Verifica tu memoria RAM libre arriba a la izquierda.
  • Usa el botón "Restaurar" cuando termines tu sesión de juego o streaming.
¿Qué ofrece ZenDash?

Automatizaciones agresivas y eficaces

🎮

Modo Gaming

Cierra navegadores y sube prioridad de GPU sacrificando procesos de Windows como SysMain.

📹

Modo Streaming

Suspende temporalmente sincronizaciones (OneDrive, Dropbox) dando espacio total a OBS/Twitch.

🧹

Limpieza Segura

Vacía tu de forma silenciosa %TEMP%, C:\Windows\Temp y la papelera de reciclaje.

🌐

Flush DNS

Reactiva y limpia cachés y Winsock para que el ping en juegos multijugador vuelva a su estado óptimo.

🚀

WinGet Update

No instales actualizadores. Con un botón llama a `winget` nativamente para asegurar que todo el software esté en su última versión.

🔋

Control de Energía

Alterna del perfil equilibrado al "Alto Rendimiento" justo cuando más lo necesitas.

Diseño Central

Los perfiles del panel

🟣 Streaming Mode Potencia CPU

Pensado para el multi-tarea pesado donde necesitas que OBS Studio y el streaming reciban la mayor tasa de procesamiento. Deja el audio Realtek limpio pero cierra todos tus clientes de mensajería (salvo si abres Discord expresamente) y sincronización.

🔵 Gaming Mode Potencia GPU

Para juegos AAA. Cambia las claves de registro temporalmente para priorizar la GPU en el sistema (Multimedia/SystemProfile/Tasks/Games) a 8, desactivando procesos de disco como SysMain.

🔴 Botón de Restaurar Vuelta a la calma

Todo lo que apagas se puede encender de nuevo. Regresa a una gestión de energía balanceada y levanta de nuevo la indización y SysMain en Windows para uso ofimático.
