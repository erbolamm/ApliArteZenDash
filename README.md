# ApliArteZenDash 🚀

Un panel de control ligero y eficiente creado en formato **HTA** para optimizar el rendimiento de Windows en segundos. Diseñado originalmente para exprimir al máximo equipos con hardware equilibrado (como el combo Intel i5-3470 + GTX 1650).

**Web Oficial**: [erbolamm.github.io/ApliArteZenDash](https://erbolamm.github.io/ApliArteZenDash/)

![ApliArte ZenDash Logo](docs/assets/logo.webp)

## 🛠️ Funciones Principales

- **Streaming Mode**: Cierra procesos en segundo plano (Dropbox, OneDrive, Spotify, etc.) para liberar ancho de banda y CPU para OBS.
- **Gaming Mode**: Maximiza los recursos para el juego, aumenta la prioridad de la GPU y desactiva servicios innecesarios como SysMain.
- **Mantenimiento**: Limpieza de archivos temporales (%TEMP%), vaciado de papelera y Flush DNS en un solo click.
- **Gestión de Energía**: Alterna rápidamente entre perfiles de "Equilibrado" y "Alto Rendimiento".
- **Actualización Total**: Mantiene tu software al día usando el motor de `winget`.

## 📋 Requisitos

- Windows 10 o 11.
- [App Installer (Winget)](https://apps.microsoft.com/detail/9nblggh4ns1m) instalado para las actualizaciones de apps.

## 🚀 Instalación y Uso

1. Descarga el archivo de la carpeta `src` (o clona el repositorio).
2. Haz doble clic en `OptimizerPanel.hta` (no requiere instalación).
3. ¡Haz clic en la acción que necesites!
